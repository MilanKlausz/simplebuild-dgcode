Welcome to the documentation of "dgcode", a set of `simplebuild
<https://mctools.github.io/simplebuild>`_ packages implementing a Geant4-based
simulation framework. It originated in the Detector Group at the European
Spallation Source ERIC, and is primarily intended for usage for simulation of
neutron detectors and related projects.

Note, that you can browse this documentation in order via the button marked
**"Next"** (or use your keyboards Left and Right arrow keys as a short
cut). Alternatively, you can use the menu or the search box to navigate to the
relevant parts of the documentation.
